# 💉 Vaccination System – Django Backend Project

This repository contains a **Vaccination System web application** built with **Django**, demonstrating full-stack backend development, database modeling, user management, and media handling.

The system is designed to manage **vaccination campaigns, centers, vaccines, and user profiles**, providing a real-world example of a scalable, secure backend application.

## 🧠 Key Features

### 1️⃣ User Management
- Registration and login system  
- Profile management with profile picture upload  
- Role-based access (e.g., admin, staff, user)

### 2️⃣ Vaccination Campaigns
- Create and manage vaccination campaigns  
- Track campaign start/end dates and locations  
- Assign vaccines to campaigns

### 3️⃣ Vaccination Centers
- CRUD operations for vaccination centers  
- Assign staff and manage schedules  

### 4️⃣ Vaccine Management
- Add and track vaccines, doses, and inventory  
- Link vaccines to campaigns and centers

### 5️⃣ Scheduling & Records
- Users can view vaccination schedule  
- Record vaccinations and track status  

### 6️⃣ Media & Static Handling
- Upload and serve profile images  
- Static files (CSS, JS) for frontend views  
- Template rendering for dynamic pages
